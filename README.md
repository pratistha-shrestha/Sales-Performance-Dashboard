# Sales Performance Dashboard – Power BI

##  Project Overview
This project analyzes sales performance data using Power BI to uncover trends in profitability, product performance, and regional sales. The dashboard enables data-driven decision-making through interactive visuals and KPIs.

##  Objectives
- Analyze total sales, profit, and profit margin
- Identify top-performing products and categories
- Compare regional sales and profitability

##  Dataset
- Source: Simulated retail sales dataset
- Time period: 2022–2024
- Records: 1,000 transactions

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel / CSV

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin %
- Average Order Value
- Total Orders

##  Key Insights
- Office Supplies contribute the highest share of total profit
- Technology products show strong sales but lower profit margins
- Higher average order values do not always result in higher profitability
